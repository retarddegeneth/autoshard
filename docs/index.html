> AutoShard

Autonomous B20 token risk agent on Base mainnet. Headless JSON API. No UI. No paid providers.

Base Mainnet JSON API SQLite Ledger MIT

Quick Start

git clone https://github.com/retarddegeneth/autoshard.git
cd autoshard
python3 app.py
# API on http://127.0.0.1:8080

API Reference

GET /health
{"status":"ok"}
POST /scan
{"address":"0x4200000000000000000000000000000000000006","chain":"base"}
-->
{"address":"0x...","symbol":"WETH","risk_score":92.0,"classification":"safe","factors":{...},"reasons":[...],"last_scanned":"..."}
GET /ledger
List all scanned tokens with latest risk scores.
POST /refresh
Recompute scores from stored factors without re-fetching on-chain data.

Risk Model

SAFE

Score >= 70

WARN

Score >= 35

DANGER

Score < 35

Base score: 100. Deductions apply for contract age, holder concentration, liquidity gaps, and hook patterns (pause, mint, fee setters). Full delta table is in SKILL.md in the repo.

RPC

Default: https://mainnet.base.org. Override with BASE_RPC=https://your-rpc. Send header X-Chain-ID: 0x2105 (Base) or 0x14a34 (Base Sepolia).

Data Store

SQLite file: b20.db, table tokens. Persists across restarts.

Verification

curl http://127.0.0.1:8080/health
