API Reference

Four endpoints. That's all you need.

Structured JSON responses, no bloat. Drop-in compatible with any agent, CI pipeline, or monitoring stack.

GET
/health
Returns {"status":"ok"}
POST
/scan { address: "0x...", chain: "base" }
Score a single token. Returns risk_score (0–100), classification, factor breakdown, and top-3 reasons.
GET
/ledger
Full SQLite-backed ledger. Every scanned token with its latest score and last-scanned timestamp.
POST
/refresh
Recompute scores from stored factors without hitting RPC again.
bash # health check curl http://127.0.0.1:8080/health # full scan example curl -X POST http://127.0.0.1:8080/scan \ -H 'Content-Type: application/json' \ -d '{"address":"0x4200000000000000000000000000000000000006","chain":"base"}' # export full ledger curl http://127.0.0.1:8080/ledger | jq .
Risk Model

Deterministic. Interpretable. Free.

Base score 100. Deductions for contract age, holder concentration, liquidity gaps, and admin hooks. No ML black box. Fully documented in SKILL.md.

SAFE

Score ≥ 70

WARN

Score ≥ 35

DANGER

Score < 35

Install

One command. Port 8080. Done.

Pure Python. stdlib only. No pip packages required. SQLite ledger persists across restarts.

Local

Clone and run. Points to Base mainnet by default.

git clone https://github.com/retarddegeneth/autoshard.git cd autoshard python3 app.py

Custom RPC

Override the default endpoint via environment variable.

BASE_RPC=https://your-rpc \ python3 app.py
